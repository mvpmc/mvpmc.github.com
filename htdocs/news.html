New Release - 2006-06-15

Version 0.3.0 of mvpmc has been released here.

The changes include:

  • support for UTF-8 encoded characters in mythtv
  • jpeg image scaling
  • upgrade to busybox 1.1.3
  • support for randomizing a playlist
  • support for new H1/H2 MediaMVP hardware
  • colorbar support
  • support for ntp
  • fullscreen gui for mclient several enhancements to native mp3 and ogg streaming
  • downloads default current internet radio playlists from Shoutcast
  • other popular internet formats, wma, asf, mms, real audio, and aac+ via vlc and mplayer
  • Podcast support
  • divx and flv video files from filebrowser using vnc
  • integrated web server for configuration, playlist management, and status reporting
  • works as dongle.bin replacement
  • "Emulation Mode" support for vnc-based Hauppauge protocol access to Hauppauge, gbpvr, MediaNet, MVPServer and mvpvdr.servers.

New Release - 2006-02-11

Version 0.2.1 of mvpmc has been released here.

The changes include:

  • add support for DNS
  • MythTV event handling
  • MythTV sort options: date recorded vs original airdate
  • support for latest MythTV protocol (version 26)
  • correctly letterbox widescreen content on 4x3 displays
  • play all option in the filebrowser
  • allow ip address to be changed in the settings menu
  • support for hiding MythTV recordings based on recording group
  • MythTV upcoming recordings filtering
  • MythTV recordings filter by: title, category, rec group
  • misc mclient updates
  • http playback support for audio and video in playlists
  • add a new theme: easy

New Release - 2005-11-02

Version 0.2.0 of mvpmc has been released here.

The changes include:

  • support building on ppc linux
  • support building with gcc 4.0
  • add support for mpeg transport streams
  • improve .wav file support
  • allow playback to continue outside the area of mvpmc that initiated the playback
  • add an audio playback thread to the file browser for better reliability
  • add shuffle play to the file browser
  • add mclient, a SlimServer client
  • add mythtv livetv support for multiple backends
  • add tcp receive buffer size settings for libcmyth
  • add popup menu in myth livetv to choose tuner
  • added support for ReplayTV 4K devices
  • updated ReplayTV discovery server to prevent DVArchive error messages
  • add ReplayTV relative jump forward/back
  • add initial VNC support
  • add shared memory area so settings can survive a crash
  • add config file support so settings can survive a reboot
  • add more themeable widgets
  • add theme names to theme menu
  • add one new theme: Big Orange
  • revamp the settings menu
  • improve brightness setting
  • add volume control
  • add a new 16:9 aspect ratio setting with widescreen signaling (WSS)
  • add the viewport setting to specify the viewable area of your TV
  • add ability to capture screenshots
