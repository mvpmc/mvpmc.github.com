--- title: New Release layout: post ---

Version 0.2.1 of mvpmc has been released here.

The changes include:

  • add support for DNS
  • MythTV event handling
  • MythTV sort options: date recorded vs original airdate
  • support for latest MythTV protocol (version 26)
  • correctly letterbox widescreen content on 4x3 displays
  • play all option in the filebrowser
  • allow ip address to be changed in the settings menu
  • support for hiding MythTV recordings based on recording group
  • MythTV upcoming recordings filtering
  • MythTV recordings filter by: title, category, rec group
  • misc mclient updates
  • http playback support for audio and video in playlists
  • add a new theme: easy
